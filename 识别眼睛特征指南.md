# 识别眼睛特征指南

## 1. 已完成的配置

已在 `assets/resource/base/pipeline/facial_feture/facial_feature.json` 中添加了黄色眼睛的识别配置：

```json
"精灵面部特征_黄色眼睛": {
    "recognition": "TemplateMatch",
    "template": "FacialFeatures/黄色眼睛.png",
    "threshold": 0.7,
    "roi": [
        320,
        100,
        380,
        120
    ]
}
```

## 2. 创建模板图片

要完成识别，需要创建黄色眼睛的模板图片：

1. **获取眼睛图像**：从游戏截图中提取黄色眼睛的清晰图像
2. **裁剪图像**：只保留眼睛区域，确保图像清晰且特征明显
3. **保存图片**：将图像保存为 `黄色眼睛.png`，放置到以下目录：
   ```
   assets/resource/base/image/FacialFeatures/
   ```

## 3. 配置说明

### 关键参数解析：

- **recognition**: "TemplateMatch" - 使用模板匹配算法
- **template**: "FacialFeatures/黄色眼睛.png" - 模板图片路径
- **threshold**: 0.7 - 匹配阈值，0.7表示70%以上匹配度即认为识别成功
- **roi**: [320, 100, 380, 120] - 识别区域，格式为 [x, y, width, height]
  - x, y: 识别区域左上角坐标
  - width, height: 识别区域宽度和高度

### 调整建议：

- **ROI区域**：根据实际游戏界面调整ROI坐标，确保只包含眼睛区域
- **阈值**：如果识别过于严格（经常失败），可降低阈值；如果误识别过多，可提高阈值
- **模板图片**：使用高质量、无干扰的眼睛图像作为模板

## 4. 使用方法

在Python代码中使用这个识别配置：

```python
from maa.agent import Agent

# 创建Agent实例
agent = Agent()

# 使用面部特征识别
result = agent.recognize("精灵面部特征_黄色眼睛")
if result:
    print("识别到黄色眼睛！")
    print("位置：", result.box)
else:
    print("未识别到黄色眼睛")
```

## 5. 测试与优化

1. **运行测试**：使用实际游戏界面测试识别效果
2. **调整参数**：根据测试结果调整ROI和阈值
3. **多模板支持**：如果有多种黄色眼睛变体，可以创建多个模板图片和对应的识别节点

## 6. 其他识别方法

除了模板匹配，还可以考虑：

- **FeatureMatch**: 特征点匹配，适合变形较大的图像
- **ColorMatch**: 颜色匹配，适合特定颜色的眼睛
- **NeuralNetworkClassify**: 神经网络分类，需要训练模型但识别准确率更高

根据实际需求选择合适的识别算法。
